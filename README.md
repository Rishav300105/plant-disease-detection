## 🚀 About Project

An AI-based web application that detects plant diseases from leaf images using Deep Learning.

---

## ✨ Features

* 📸 Upload plant leaf image
* 🤖 AI-based disease prediction
* ⚡ Fast and accurate results
* 🎯 Simple & clean UI

---

## 🛠️ Tech Stack

* 🐍 Python
* 🌐 Flask
* 🧠 TensorFlow / Keras
* 🎨 HTML, CSS

---

## 📂 Project Structure

```
plant-disease-detection/
│── app.py
```

## ▶️ How to Run

```bash
git clone https://github.com/Rishav300105/plant-disease-detection.git
cd plant-disease-detection
pip install -r requirements.txt
python app.py
```

## ⚠️ Important Note

Model file is not included due to size limits.
It will be downloaded automatically when you run the project.
